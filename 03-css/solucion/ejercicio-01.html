Algunas propiedades comunes

Propiedad Descripción Ejemplo
color Indica el color de fuente { color: red; } { color: #FF0000; } { color: rgb(255, 0, 0); }
margin Indica la separación del elemento contra otros elementos. Se especifica en el orden: top right bottom left { margin: 10px 5px 8px 3px; }

Donde:
top = 10px
right = 5px
bottom = 8px
left = 3px
También se puede indicar individualmente cada lado { margin-top }
padding Indica la separación del borde del elmento contra su contenido { padding: 10px; } Al igual que margin, se puede indicar todos los lados en una sola propiedad o definirlos de forma separada
